# Final API

API REST desarrollada con .NET Core 9 con CI/CD en Azure DevOps.

## Requisitos
- .NET Core 9 SDK
- Visual Studio Code o Visual Studio

## Ejecución Local
```bash
dotnet build
dotnet run --project ./src/FinalAPI.Api
```

## Documentación API
Acceder a Swagger: http://localhost:5094/swagger

## CI/CD
Este proyecto usa Azure DevOps para CI/CD automático.
